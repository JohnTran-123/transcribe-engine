# Meeting Prompt Overlay (optional)

> Extra rules appended to the refine pass. Use this for output customization
> beyond the default Decisions / Actions / Debates / Topics sections.

## Additional output sections
After the Topics section, add:

**## Risks Surfaced**
For each risk raised: what the risk is, who raised it, any mitigation discussed.

**## Open Questions**
Anything raised but not answered, with the asker named.

## Formatting rules
- Render code identifiers in `backticks`
- Render product names in **bold**
- Render lab compound IDs (e.g. COMPOUND-7) exactly as in the glossary
